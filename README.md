# LinguaLearn

A small vocabulary learning application built with HTML, CSS and Vanilla JavaScript.

## Features

- Add new vocabulary
- Delete vocabulary
- Flashcards system
- Mark words as known
- Search functionality
- LocalStorage persistence
- Responsive design

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage

## Project Structure

```bash
assets/
css/
js/
index.html
## Run Locally

```bash
npm install
npm run dev
```
